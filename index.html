COMANDOS
  connect        conectarse a una red Wi-Fi
  help           mostrar esta ayuda
  hotspot        crear un punto de acceso Wi-Fi (hotspot)
  list           listar las redes Wi-Fi disponibles
  rescan         volver a escanear las redes Wi-Fi disponibles
  show-password  mostrar la contraseña de la red Wi-Fi actual

    Audio Debian

    Uso: wpctl [OPCIÓN…] COMANDO [OPCIONES_DE_COMANDO]

Comandos:
  status               mostrar estado
  get-volume ID        obtener el volumen del ID
  inspect ID           inspeccionar el ID
  set-default ID       establecer ID como predeterminado
  set-volume ID VOL[%][-/+]  establecer volumen del ID (ej. 0.5 o 50%+)
  set-mute ID 1|0|toggle     silenciar ID (1=sí, 0=no, toggle=alternar)
  set-profile ID ÍNDICE      establecer perfil del ID
  set-route ID ÍNDICE        establecer ruta del ID
  clear-default [ID]         quitar predeterminado [ID]
  settings [CLAVE] [VALOR]   ajustes de WirePlumber
  set-log-level [ID] NIVEL   establecer nivel de registro de logs
